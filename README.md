# 🍽️ Smart Cafe POS System (Web App)
A modern, beautiful, and interactive Restaurant Management System built using Python and Streamlit.

### ✨ Features:
* **Interactive Digital Menu:** Click to add items to the cart.
* **Live Order Cart:** See your selected items in real-time.
* **Automated Billing:** Automatically calculates Subtotal, 5% GST, and the Final Amount.
* **Clean UI/UX:** Built with Streamlit for a modern web look.
